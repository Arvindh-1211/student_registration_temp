# student-registrarion

## Run the container
1. Change the .env file in both frontend and backend
```
    docker-compose up -d
```

## Frontend
1. Change the .env file

```
    git pull
    cd ./frontend
    npm i
    npm run build
```

## Frontend dev
```
    git pull
    cd ./frontend
    npm i
    npm start
```

## Backend
1. Change the .env file
```
    git pull
    cd ./backend
    npm i
    npm start
```

## Backend dev
```
    git pull
    cd ./backend
    npm i
    npm run dev
```
